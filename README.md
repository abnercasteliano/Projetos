## [Previsão do preço de carros]

O objetivo é implementar um algoritmo de Machine Learning adequado e responder algumas questões de negócio:

1)	Prever o preço de um Mercedez Benz com 4 anos de idade e quilometragem rodada de 45000.
2)	Prever o preço de uma BMW X5 com 7 anos de uso e quilometragem de 86000.
3)	Me diga qual a precisão do modelo em porcentagem %.
