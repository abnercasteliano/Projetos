## [Churn Classifier]()

O objetivo do projeto é fazer uma análise exploratória dos dados da empresa, afim de identificar o que causa a saída de alguns clientes. Após a análise, foi criado um modelo preditivo que classifica os clientes em churner(cliente que vai deixar a empresa) ou não churner.
