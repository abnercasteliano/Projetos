# Projetos

Repositório contendo alguns projetos de Machine Learning e Data Science.

## [Vehicle Price prediction](https://github.com/abnercasteliano/vehicle-price-prediction/tree/026a4e89202ffa1a8cad2121bb5db0517a6f64cd)

O neste projeto foi criado um modelo de Machine Learning capaz de prever o preço de venda do veículo, de acordo com algumas características do carro como ano, quilometragem rodada, etc.

## [Churn Classifier](https://github.com/abnercasteliano/customer-churn/tree/4c139cd6986bb0bdd1620bfa9ad3f66aa91d6dd8)

O objetivo do projeto é fazer uma análise exploratória dos dados da empresa, afim de identificar o que causa a saída de alguns clientes. Após a análise, foi criado um modelo preditivo que classifica os clientes em churner(cliente que vai deixar a empresa) ou não churner.

## [Bayesian-A-B-testing](https://github.com/abnercasteliano/Bayesian-A-B-testing/tree/8e97a771c4541b1168f84361bc1b6aaa16bf6682)

O objetivo é fazer um teste baseado no teorema bayesiano, para comparar a taxa média de conversões entre duas páginas diferentes.
